Step:1 =>Install Node js Latest Version (https://reactnative.dev/docs/environment-setup)
Step:2 => Install Android Studio (https://developer.android.com/studio/index.html)
Step:3 =>Install the Android SDK and setup android Virtual devices
Step:4 =>Configure the ANDROID_HOME environment variable
Step:5 =>install jdk

Step:6 =>open our project directory Amezon and run cmd for this directory
Step:7 =>then run the command using cmd "npx react-native start"
Step:8 =>then run the command using cmd "npx react-native run-android"

Step:9 => Enjoy
